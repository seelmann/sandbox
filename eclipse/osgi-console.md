
Open _Console_ view, start _Host OSGi Console_.

    # Get Help
    osgi> help
    
    # List all bundles
    osgi> lb
    
    # List bundles matching a substring
    osgi> lb collections
    osgi> lb "Apache Directory Studio"
    
    # Start a bundle
    osgi> start 920
    
    # Uninstall a bundle
    osgi> uninstall 39
