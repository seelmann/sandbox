Dotfiles
========

## Requirments

Install tools:

    apt install vim zsh git tmux

Create local bin:

    mkdir ~/bin

Install `oh-my-zsh`.

## Keyboard

Keyboard layout: International AltGr with keys

Map Thinkpad T420 prev/next to pg up/down:

    xmodmap -e 'keycode 166=Prior'
    xmodmap -e 'keycode 167=Next'
